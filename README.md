# Python-port-scanner
A multi-threaded TCP port scanner with banner grabbing capabilities, built with Python's standard library for performance and portability.
**License:** MIT  
**Python Version:** 3.6+

---

## 🚀 Features

- **Multi-threaded Scanning:** Concurrent port scanning for faster results  
- **Banner Grabbing:** Retrieves service banners from open ports  
- **Service Detection:** Identifies common services by port number  
- **Flexible Port Specification:** Supports ranges and comma-separated lists  
- **Timeout Control:** Configurable connection timeouts  
- **Multiple Output Formats:** JSON and CSV export options  
- **Verbose Mode:** Detailed output including banner information  
- **Error Handling:** Handles common network-related errors gracefully  
- **DNS Resolution:** Supports both IP addresses and hostnames  

---
